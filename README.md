# `@simplefocus/prettier-config`

> Simple Focus base prettier configuration.


## Usage

__Install:__

```bash
$ yarn add --dev @simplefocus/prettier-config
```

__Update `package.json`:__

```jsonc
{
  // ...
  "prettier": "@simplefocus/prettier-config"
}
```
